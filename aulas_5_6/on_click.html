Usando onclick do JS


O evento onclick chama a nossa função em JS dentro das tags scripst
isso ocorre quando o usuário clicar no botão!

Quando você usa innerHTML em um elemento HTML,
você pode obter o conteudo HTML dentro desse elemento de página.
Aqui esta um exemplo básico de como usar o InnerHTML com JS.
Código Fonte:
function my_function() {
document. getElementById("elemento_id" .innerHTML = "Hello World"
}
